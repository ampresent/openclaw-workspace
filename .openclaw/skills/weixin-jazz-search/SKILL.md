# 微信公众号上海爵士演出搜索

## 功能
- 搜索关键词：`上海 爵士 演出 2026`（**注意：关键词中间有空格**）
- 搜索源：搜狗微信搜索 (https://weixin.sogou.com/)
- 过滤条件：发布后 7 天内
- 验证规则：确认是上海爵士音乐演出（排除舞蹈、校考等）
- 输出格式：Markdown 表格
- 搜索范围：前 3 页（约 30 条结果）

## 使用方式

### 手动调用
```python
# 使用 browser tool 访问搜狗微信搜索
browser(action="open", targetUrl="https://weixin.sogou.com/weixin?type=2&query=上海 爵士 演出 2026")

# 获取搜索结果
browser(action="snapshot", refs="aria")

# 提取结果并过滤
# 1. 解析每条结果的标题、公众号、发布时间、链接
# 2. 过滤 7 天内的内容
# 3. 验证是否真的是爵士演出
# 4. 整理成表格
```

### 定时任务
Cron Job 每天 20:00 自动执行，输出到：
1. 聊天消息
2. 文件：`data/weixin-jazz-search/YYYY-MM-DD.md`

## 验证规则

内容必须同时满足：
- ✅ 包含：`爵士` AND (`演出` OR `音乐会` OR `音乐节` OR `live` OR `Live` OR `专场` OR `票价`)
- ✅ 包含地点：`上海` OR 上海 venue（JZ Club、林肯爵士、西岸大剧院、长滩音乐厅等）
- ❌ 排除：`爵士舞蹈 `、` 爵士舞 `、` 街舞 `、` 校考 `、` 舞蹈生`

## 输出格式

```markdown
# 上海爵士演出周报 (YYYY-MM-DD)

| 演出名称 | venue/地点 | 时间 | 来源公众号 | 发布日期 | 文章链接 |
|----------|------------|------|------------|----------|----------|
| 爵士狂欢即将开启，解锁上海春日的浪漫打开方式 | 上海西岸大剧院 | 4/30-5/5 | 徐汇文旅 | 2026-03-18 | [链接] |
```

## 错误处理

| 错误 | 处理方式 |
|------|----------|
| 搜索无结果 | 输出"今日无符合条件的上海爵士演出信息" |
| 浏览器失败 | 重试 2 次，仍失败则报错 |
| 保存失败 | 至少输出到聊天 |
| 触发验证码 | 暂停任务，通知用户手动处理 |
