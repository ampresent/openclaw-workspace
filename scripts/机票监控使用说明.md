## 🔧 使用方法

### 方法 A: 使用 OpenClaw 自动化（推荐）

由于浏览器服务偶尔不可用，定时任务会在浏览器可用时自动执行查询。

**查看 Cron 任务状态：**
```bash
openclaw cron list
```

**手动触发一次查询：**
```bash
openclaw cron run be8b6d48-d41a-4c9d-bb1e-0d81643cb554
```

### 方法 B: 本地 Python 脚本（备用）

如果 OpenClaw 浏览器服务不稳定，可以在本地运行 Python 脚本：

**1. 安装依赖：**
```bash
pip install playwright matplotlib
playwright install chromium
```

**2. 首次登录（扫码）：**
```bash
cd /home/admin/openclaw/workspace
python scripts/fliggy_login.py
```

**3. 执行查询：**
```bash
python scripts/flight_monitor_full.py
```

**4. 查看结果：**
```bash
# 查看价格记录
cat data/flights/shanghai-chengdu-prices.csv

# 查看图表（如果有图形界面）
open data/flights/price-trend.png  # macOS
xdg-open data/flights/price-trend.png  # Linux
```

### 方法 C: 手动查询 + 自动记录（最简单）

**1. 手动查询机票：**
- 打开飞猪官网：https://www.fliggy.com
- 搜索：上海→成都，2026-05-15
- 筛选：18:00-23:00 起飞
- 找到最低价格

**2. 使用手动模式记录：**
```bash
python scripts/flight_monitor_full.py --manual
```
然后输入价格和航班信息即可。

## ⚠️ 注意事项

1. **浏览器服务**：OpenClaw 浏览器服务偶尔会超时，如遇问题请重启 Gateway：
   ```bash
   openclaw gateway restart
   ```

2. **登录状态**：Cookies 会保存在 `scripts/cookies.json`，如果登录过期请重新运行：
   ```bash
   python scripts/fliggy_login.py
   ```

3. **图表生成**：需要安装 matplotlib，如果不需要图表可以跳过。

4. **时间范围**：当前配置为查询 18:00-23:00 起飞的航班，如需修改请编辑 `scripts/flight_monitor_full.py` 中的 `SEARCH_CONFIG`。
