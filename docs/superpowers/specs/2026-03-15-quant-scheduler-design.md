## 1. 系统概述

为 A 股量化交易系统添加周期运行功能，每日自动获取小微盘股数据、生成交易信号，并保存到本地文件。

### 1.1 设计目标

- **自动化**: 每日定时运行，无需人工干预
- **小微盘聚焦**: 筛选市值最小的 200-500 只股票
- **本地存储**: 买卖信号保存到 JSON/CSV 文件
- **可靠稳定**: 错误处理、日志记录、失败重试

---

## 2. 核心需求

| 项目 | 选择 |
|------|------|
| 运行频率 | 每日运行（交易日） |
| 运行时间 | 20:00（晚上 8 点） |
| 股票池 | 沪深市场市值最小的 200-500 只小微盘股 |
| 输出方式 | 生成报告 + 买卖信号本地文件 |
| 存储格式 | JSON + CSV |
| 信号类型 | 买入和卖出信号都保存 |

---

## 3. 技术选型

| 模块 | 选择 | 说明 |
|------|------|------|
| 定时任务 | cron | Linux 系统定时器，简单可靠 |
| 信号存储 | Python json/csv | 标准库，JSON+CSV双格式 |
| 小微盘筛选 | AkShare | 获取股票列表和市值数据 |
| 日志记录 | Python logging | 标准库，支持文件输出 |

## 6. 模块设计

### 6.1 股票池筛选模块 (stock_pool/)

**职责**: 筛选小微盘股票，生成候选股票列表

**主要函数**:
```python
def get_all_stocks() -> pd.DataFrame
    """获取全市场股票列表和市值数据"""

def filter_small_cap(top_n: int = 300) -> List[str]
    """筛选市值最小的 N 只股票"""

def save_watchlist(stocks: List[str], filepath: str) -> None
    """保存股票列表到文件"""

def load_watchlist(filepath: str) -> List[str]
    """从文件加载股票列表"""
```

**数据字段**:
- code: 股票代码
- name: 股票名称
- market_cap: 总市值
- float_cap: 流通市值

### 6.2 定时任务模块 (scheduler/)

**职责**: 定时任务入口，协调整个流程

**主要函数**:
```python
def run_daily_task() -> Dict
    """执行每日任务"""
    
def is_trading_day() -> bool
    """判断是否为交易日"""
```

**工作流程**:
1. 检查是否为交易日
2. 获取/更新股票池
3. 批量获取股票数据
4. 生成交易信号
5. 发送信号通知
6. 记录运行日志

### 6.3 信号存储模块 (notifier/)

**职责**: 保存交易信号到本地文件（JSON + CSV 双格式）

**主要函数**:
```python
def save_signals(signals: List[Dict], 
                 output_dir: Path,
                 date: str = None) -> Dict[str, Path]
    """保存信号到 JSON 和 CSV 文件"""

def save_signals_json(signals: List[Dict], output_dir: Path) -> Path
    """保存信号到 JSON 文件"""

def save_signals_csv(signals: List[Dict], output_dir: Path) -> Path
    """保存信号到 CSV 文件"""

def format_signals_text(signals: List[Dict]) -> str
    """生成文本格式的信号摘要（控制台输出）"""
```

**输出格式**:
- JSON: 完整数据结构，包含摘要和信号列表
- CSV: 表格格式，便于 Excel 打开
- 文本：控制台打印的格式化输出

## 8. 信号文件设计

### 8.1 JSON 格式

```json
{
  "date": "2026-03-15",
  "generated_at": "2026-03-15 20:05:00",
  "summary": {
    "total": 4,
    "buy": 3,
    "sell": 1
  },
  "signals": [
    {
      "type": "buy",
      "code": "300XXX",
      "name": "XXX",
      "price": 12.34,
      "strength": "强 (3/3)"
    }
  ]
}
```

### 8.2 CSV 格式

```csv
date,type,code,name,price,strength

2026-03-15,sell,000XXX,XXX,23.45，中 (2/3)

0 20 * * * cd /home/admin/openclaw/workspace/quant-system && python3 scheduler/run_daily.py >> logs/cron.log 2>&1

## 10. 错误处理

### 10.1 异常情况

| 异常 | 处理方式 |
|------|----------|
| 网络错误（数据获取失败） | 重试 3 次，失败后记录日志 |
| 股票数据为空 | 跳过该股票，继续处理下一个 |
| 邮件发送失败 | 记录日志，不中断流程 |
| 非交易日 | 直接退出，记录"非交易日" |

### 10.2 日志格式

```
2026-03-15 20:00:00 INFO - 开始执行每日任务
2026-03-15 20:00:01 INFO - 检查交易日：是
2026-03-15 20:00:03 INFO - 获取股票列表：成功 (4800 只)
2026-03-15 20:00:05 INFO - 筛选小微盘股：300 只
2026-03-15 20:05:00 INFO - 获取数据完成：298 只（2 只失败）
2026-03-15 20:10:00 INFO - 生成信号：买入 3，卖出 1
2026-03-15 20:12:00 INFO - 邮件发送：成功
2026-03-15 20:12:01 INFO - 任务完成
